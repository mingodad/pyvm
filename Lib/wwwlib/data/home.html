Help
  • Right click on "Back" for jump back
  • Holding CTRL while using the scroll wheel, scrolls faster
  • Back using "BACKSPACE" restores the pointer
  • "TAB" switches tabs
  • SHIFT+click on a link, downloads and saves the link in the "Download area"
  • press '1' for plain HTML mode, '8' for attempt to full CSS (pretty much broken)
